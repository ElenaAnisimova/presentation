OAuth

OAuth stands for Open Authorization

OAuth versions

  • OAuth 1.0a
  • OAuth 2.0

OAuth process

1. App requests authorization from User

2. User authorizes App and delivers proof

3. App presents proof of authorization to server

4. Token can only access what the User authorized for the App
